# 更新日志

本项目所有重要变更都会记录在此文件，遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## v0.2 - 2026-06-20

### 新增
- 后台服务器列表支持**拖拽排序**，前台首页（卡片 / 列表）按自定义顺序展示；新增服务器自动排到末尾。
- 添加服务器表单的「国旗代码」标注**「不填则自动」**：留空时使用 agent 通过 GeoIP 自动识别的国家国旗。

### 优化
- 服务器详情页实时小卡顺序调整为 **系统负载 / 内存 / 硬盘 / 总流量**。
- 历史图表重新编排：CPU 独立成图；新增**「内存 / 交换」**图（交换分区由原硬盘图并入）；硬盘独立成图；「内存」图与「网络速率」图位置对调。

## v0.1.5 - 2026-06-20

### 修复
- 源站统一下发缓存头，修复发版后浏览器仍加载旧版页面的问题。
- 安装脚本 systemd 改用 `restart` 替代 `enable --now`，修复删号重装后 agent 不上线。

## v0.1.4 - 2026-06-20

### 新增
- agent 自动识别公网 IP、国旗与虚拟化类型。

### 修复
- 修复面板六处问题。

### 文档
- 补充 nginx 反代示例配置与部署说明。

## v0.1.3 - 2026-06-20

### 修复
- 安装脚本在 root 且无 sudo 的精简系统上直接安装，修复 `sudo: command not found`。

## v0.1.2 - 2026-06-20

### 新增
- 登录支持自定义用户名（默认 `admin`），密码连续错误 3 次锁定来源 IP 30 分钟。

## v0.1.1 - 2026-06-20

### 新增
- `moss.sh` 一键部署 / 管理脚本（安装、更新、卸载，自动生成管理员密码），并安装为全局 `moss` 命令。

### 修复
- Docker 多架构镜像（amd64 + arm64），修复 arm64 服务器拉取失败。

## v0.1.0 - 2026-06-20

- 首个版本：轻量自托管服务器监控，包含核心采集与展示、离线 / 负载告警、Docker 与 CI。
