# 更新日志

本项目所有重要变更都会记录在此文件，遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## v0.2.1 - 2026-06-21

### 新增
- 后台服务器列表新增 **IPv6 展示**：agent 同时探测公网 IPv4 / IPv6，IP 列上下两行分别显示（无 IPv6 的主机仅显示 IPv4），显隐与掩码沿用原逻辑。升级后需**重新部署 agent** 才会上报 IPv6。

### 优化
- **历史采样间隔改为秒级、可连续设置**（5 ~ 3600 秒，默认 10 秒）：时间轴可平滑缩放到秒级，不再受 5 分钟粒度下限限制；历史默认保留期下调为 7 天。升级时旧库自动迁移（原分钟值 ×60，落库节奏不变）。
- 服务器详情页「总流量」卡片重排为 **实时网速 / 总流量** 双栏。
- 主页卡片与详情页的**实时网速数字改为逐位跳动**：每位数字独立成元素，只有发生变化的那一位滚动更新、未变的位不动，不再整段数字重绘。

### 修复
- 延迟监控的「实时」选项现按**实时上报间隔**自动轮询刷新，修复选中实时却始终不更新的问题。

## v0.2 - 2026-06-20

### 新增
- 后台服务器列表支持**拖拽排序**，前台首页（卡片 / 列表）按自定义顺序展示；新增服务器自动排到末尾。
- 添加服务器表单的「国旗代码」标注**「不填则自动」**：留空时使用 agent 通过 GeoIP 自动识别的国家国旗。

### 优化
- 服务器详情页实时小卡顺序调整为 **系统负载 / 内存 / 硬盘 / 总流量**。
- 历史图表重新编排：CPU 独立成图；新增**「内存 / 交换」**图（交换分区由原硬盘图并入）；硬盘独立成图；「内存」图与「网络速率」图位置对调。
